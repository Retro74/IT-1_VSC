Sjekk konsollet

Trykk F12 for å få frem konsollet 


let alder = 17
console.log(`Alder en er nå ${alder}, alder er definert som ${typeof(alder)}`)
> Alder en er nå 17, alder er definert som number

alder = alder +1
console.log(`Ett år senere er alderen ${alder}, alder er nå definert som ${typeof(alder)}`)
> Ett år senere er alderen 18, alder er nå definert som number

alder += 1
console.log(`Ett år senere er alderen ${alder}, alder er nå definert som ${typeof(alder)}`)
console.log(typeof(alder))
> Ett år senere er alderen 19, alder er nå definert som number

alder ++
console.log(`Ett år senere er alderen ${alder}, alder er nå definert som ${typeof(alder)}`)
> Ett år senere er alderen 20, alder er nå definert som number

alder += "1"
console.log(`Legg til '1' år til, så er alderen ${alder}, alder er nå definert som ${typeof(alder)}`)
> Legg til '1' år til, så er alderen 201, alder er nå definert som string
